# s21_stingplus

Первый учебный **групповой** проект, выполненный в Школе 21.  
Суть проекта — реализация собственной версии стандартной библиотеки `string.h` с расширениями, включая частичную реализацию `sprintf` и `sscanf`и специальных функций обработки строк.
## О проекте

Проект направлен на изучение работы со строками в C, управления памятью и реализации форматированного ввода/вывода. В рамках работы была полностью воссоздана логика стандартных строковых функций, а также реализованы функции форматирования.

---
## Реализованный функционал

### Функции `string.h`

- функции работы с памятью: `memcpy`, `memset`, `memcmp`, `memchr`
- функции работы со строками: `strlen`, `strcpy`, `strncpy`, `strcat`, `strncat`
- поиск и сравнение: `strchr`, `strrchr`, `strstr`, `strncmp`, `strpbrk`, `strcspn`
- разбиение строки: `strtok`
- обработка ошибок: `strerror`
---
### Дополнительные функции

- `sprintf` (отправляет форматированный вывод в строку)
- `sscanf` (считывает форматированный ввод из строки)

**Специальные функции обработки строк:**
- `to_upper` (преобразует строку в верхний регистр)
- `to_lower` (преобразует строку в нижний регистр)
- `insert` (вставляет одну строку внутрь другой в указанное место)
- `trim` (убирает заданные символы в начале и конце строки)

- Язык: **C**, стандарт 11

- Стандарт: **POSIX.1-2017**
- Стиль: **Google Style**

- Сборка: **Makefile** с целью `s21_string.a` для сборки статической библиотеки с реализацией функций `s21_string.h`
- Тестирование: **Unit-тесты**, библиотека Check
- Покрытие: **gcov-отчёт в HTML формате**
